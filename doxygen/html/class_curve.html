Detailed Description

Curve osztály.

Egy vector<Arrow>-ban tárolja a vezeték kis egyenes darabjait. Minden darabhoz lehet rendelni egy áramerősséget a current vectorban. A with_current kapcsoló arra lenne jó, ha a vezetőben különböző erősségű áramok folynának, akkor minden kis darabban folyhatna akár más-más áram. De ez a funkció nem lett implementálva.
